Important Notice - Point Deduction System

Please be aware: Using the online document request feature will result in automatic point deductions from your barangay citizen scoring system. Points may be deducted for each request submitted. Consider visiting the barangay office in person if you prefer not to have points deducted from your account.

By proceeding with this online request, you acknowledge and accept the point deduction policy.

List of Valid IDs

Please ensure you submit a clear copy of at least one of the following valid IDs:

  • Passport
  • Driver's License
  • Social Security System (SSS) ID
  • Government Service Insurance System (GSIS) ID
  • Unified Multi-Purpose ID (UMID)
  • Professional Regulation Commission (PRC) ID
  • Philippine National Police (PNP) Firearm License
  • Postal ID
  • Voter's ID
  • Integrated Bar of the Philippines (IBP) ID
  • Senior Citizen's ID
  • Persona with Disability (PWD) ID
  • Other valid government-issued IDs
